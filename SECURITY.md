# Security policy

If you discover a security issue, please report it privately.

## How to report

- Open a private GitHub security advisory in this repository.
- Include steps to reproduce, affected versions, and any relevant logs or proofs of concept.
- Please avoid filing public issues for potential vulnerabilities.

## Expectations

This is a hobby project; responses are best-effort and no SLA is guaranteed. You may
be asked for more detail so fixes can be validated and tested.
